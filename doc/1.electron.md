### 版本 本地版本

```sh
$ node -v
v22.14.0
$ npm -v
11.4.2
```

### 初始化 npm 项目

```sh
mkdir XXX && cd XXX
npm init
```

### 安装 electron

```sh
npm install electron --save-dev
```

### 添加 .gitignore 文件

在网址中打开直接复制，比较全面

```
https://github.com/github/gitignore/blob/main/Node.gitignore
```

### 添加运行指令

在package.json 中 main 要 设置当前的入口文件

```json
"main": "main.ts",
```

在 package.json 中的 scripts 添加指令

```json
"start": "electron ."
```

### 运行

```sh
npm run start
```
